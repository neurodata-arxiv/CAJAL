CAJAL Matlab Toolbox: Collaborative Annotation for Joint Analysis of Large Data


The CAJAL toolbox (or API) provides functionality to interact with NeuroData image and annotation databases, including querying raw image data, annotation data, and RAMON objects. Complex HTTP queries are wrapped in easy to use Matlab code - helper functions and example files are provided. New users may wish to begin with the Getting Started Script.

[ZIP] [TAR.GZ]

Please see the tutorials section for information about getting and putting data, querying for data, RAMONifying your dataset, and generating graphs. CAJAL implements the RAMON data standard explained here and here.
